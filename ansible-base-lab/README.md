# Quick start

## Clone repository

Clone this git repository:

`git clone https://github.com/aernesto24/solutions-docker-labs.git`

## Build images and run containers

`cd solutions-docker-labs/ansible-base-lab`

And then run:

`docker-compose up -d --build`
